This readme file will guide you through the process of running the provided Python code and setting the required environmental variables. Please follow the steps below to ensure successful execution.

Before proceeding, make sure you have the following:
1. Python installed on your machine.
2. An AWS account with access key ID and secret access key. (If you don't have one, please create an account and generate the necessary credentials)

To set the required environmental variables, follow these steps:
1. Open a terminal or command prompt on your machine.
2. Set the `aws_access_key_id` environmental variable by running the following command:-
	`set aws_access_key_id=your_access_key_id` 
(Replace `your_access_key_id` with your actual AWS access key ID)
3. Set the `aws_secret_access_key` environmental variable by running the following command:
	`set aws_secret_access_key=your_secret_access_key`
(Replace `your_secret_access_key` with your actual AWS secret access key)

To run the provided Python code, follow these steps:

1. Open your preferred text editor or IDE.
2. Create a new file and copy the provided code into it.
3. Save the file with a `.py` extension. For example, `my_script.py`.
4. Open a terminal or command prompt on your machine.
5. Navigate to the directory where your Python file is located using the `cd` command.
6. Once you are in the correct directory, type `python` followed by the name of your Python file. For example, `python my_script.py`.
7. Press Enter to execute the command, and the code will run, providing the desired output.

By following the steps outlined in this readme file, you should now be able to set the required environmental variables and run the provided Python code. If you encounter any issues or have further questions, please don't hesitate to ask.
